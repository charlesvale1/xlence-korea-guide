# 국내은행 입금 가이드

## Step 1–2 — 입금하기 → 온라인 뱅킹 송금 선택
1. 대시보드 → **'입금하기'** 클릭
2. **'온라인 뱅킹 송금'** 선택

## Step 3 — 금액 입력
- 통화: **KRW** · 입금 금액 입력 후 **'전송'** 클릭

## Step 4 — 이용약관 동의
체크박스 선택 후 **'다음'** 클릭

## Step 5 — 가상계좌 입금
{% hint style="danger" %}
⏱️ 화면에 표시된 가상계좌로 **30분 이내** 입금 완료 필수
{% endhint %}

## Step 6–7 — 입금 확인
| 상태 | 의미 |
|------|------|
| 🔄 검토 대기 중 | 처리 중 |
| ✅ 완료 | 입금 반영 완료 |

💬 입금 지연 시 [@kr_helpdesk](https://t.me/kr_helpdesk) 문의
{% endhint %}
