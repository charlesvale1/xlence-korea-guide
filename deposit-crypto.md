# 암호화폐 입금 가이드

## Step 1–2 — 입금하기 → 디지털 자산 선택
1. 대시보드 → **'입금하기'** 클릭
2. **'디지털 자산'** 선택

## Step 3 — 입금 정보 설정
| 항목 | 권장 설정 |
|------|:--------:|
| Currency | **USDT** |
| Network | **TRC20** |

SUBMIT 클릭 후 **지갑 주소를 복사**하세요.

{% hint style="warning" %}
📋 지갑 주소를 메모장에 저장해 두면 실수를 방지할 수 있습니다.
{% endhint %}

## Step 4 — 거래소에서 전송
OKX · 바이낸스 · 바이비트 · 비트겟에서 복사한 주소로 USDT 전송

{% hint style="danger" %}
⚠️ 네트워크 반드시 **TRC20** 선택 — 다른 네트워크 선택 시 자산 손실 가능
{% endhint %}

## Step 5 — 입금 확인
거래 내역 상태가 **'완료'** 로 변경되면 입금 반영 완료.

{% hint style="info" %}
💬 문의: [@kr_helpdesk](https://t.me/kr_helpdesk)
{% endhint %}
