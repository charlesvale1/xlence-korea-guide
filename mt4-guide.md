# MT4 사용 방법

{% hint style="info" %}
📌 MT4 앱 사용 전 **거래 계좌 생성**이 완료되어 있어야 합니다.
거래 계좌 번호와 비밀번호를 미리 준비해주세요.
{% endhint %}

---

## Step 1 — MT4 앱 다운로드

앱스토어 또는 구글플레이에서 **MetaTrader 4** 검색 후 설치

![](https://raw.githubusercontent.com/charlesvale1/xlence-korea-guide/main/image/xlence_mt4_1.png)

| 항목 | 내용 |
|------|------|
| 앱 이름 | MetaTrader 4 |
| 개발사 | MetaQuotes |
| 카테고리 | 금융 |
| 평점 | ★★★★★ 4.8 |

- **iOS**: [App Store에서 다운로드](https://apps.apple.com/app/metatrader-4/id496212596)
- **Android**: [Google Play에서 다운로드](https://play.google.com/store/apps/details?id=net.metaquotes.metatrader4)

---

## Step 2 — 브로커 서버 검색

앱 설치 후 실행 → **'계좌로 로그인'** 클릭 → **'트레이딩 서버 선택'** 화면에서:

![](https://raw.githubusercontent.com/charlesvale1/xlence-korea-guide/main/image/xlence_mt4_2.png)

브로커 검색창에 **`Xlence`** 입력 후 검색

{% hint style="warning" %}
⚠️ 브로커를 찾지 못할 경우 하단 **'브로커를 찾지 못했습니까?'** 버튼을 클릭하여 수동으로 서버를 입력할 수 있습니다.
{% endhint %}

---

## Step 3 — 서버 선택 및 로그인

검색 결과에서 **`Xlence-Real6`** (Tradeco Ltd.) 선택

![](https://raw.githubusercontent.com/charlesvale1/xlence-korea-guide/main/image/xlence_mt4_4.png)

| 항목 | 입력 내용 |
|------|---------|
| 로그인 | **이전에 생성한 Xlence 거래 계좌 번호** 입력 |
| 비밀번호 | **거래 계좌 생성 시 설정한 비밀번호** 입력 |
| 비밀번호 저장 | ✅ 체크 권장 |

입력 완료 후 **'로그인'** 버튼 클릭

{% hint style="info" %}
💡 거래 계좌 번호는 GitBook → [거래 계좌 생성](create-account.md) 페이지에서 확인 가능합니다.
{% endhint %}

---

## Step 4 — 로그인 완료 및 기본 화면 안내

![](https://raw.githubusercontent.com/charlesvale1/xlence-korea-guide/main/image/xlence_mt4_5.png)

| 💼 트레이드 | 현재 보유 포지션 및 주문 내역 |
| 📰 뉴스 | 글로벌 금융 뉴스 |
| 📬 메시지 | 브로커 공지 및 메시지 |

## Step 5 — 매수/매도 주문하기

![](https://raw.githubusercontent.com/charlesvale1/xlence-korea-guide/main/image/xlence_mt4_6.png)

**'시세'** 탭에서 거래할 통화쌍 선택 → 길게 누르기 → **'새 주문'** 클릭

| 항목 | 설명 |
|------|------|
| 심볼 | 거래할 통화쌍 (예: EURUSD, GBPUSD) |
| 볼륨 | 거래량 (0.01 = 마이크로랏, 0.1 = 미니랏, 1.0 = 스탠다드랏) |
| Stop Loss | 손절 가격 설정 (리스크 관리 필수) |
| Take Profit | 익절 가격 설정 |
| 매수 (Buy) | 가격 상승 예상 시 클릭 |
| 매도 (Sell) | 가격 하락 예상 시 클릭 |

{% hint style="danger" %}
⚠️ **반드시 Stop Loss(손절)를 설정하세요.** 설정하지 않으면 예상치 못한 큰 손실이 발생할 수 있습니다.
{% endhint %}
