# MT4 사용 방법

{% hint style="info" %}
📌 MT4 사용 전 거래 계좌 생성이 완료되어 있어야 합니다.
{% endhint %}

## Step 1 — MT4 다운로드
[xlence.com/ko/platforms](https://www.xlence.com/ko/platforms/) 에서 **MetaTrader 4** 다운로드

## Step 2 — 로그인 정보 입력
| 항목 | 내용 |
|------|------|
| Server | 계좌 생성 시 제공된 서버명 |
| Login | 거래 계좌번호 |
| Password | 계좌 비밀번호 |

## Step 3 — 거래 시작
- **Market Watch**: 통화쌍 목록
- **New Order (F9)**: 주문 입력
- **Stop Loss / Take Profit**: 리스크 설정
