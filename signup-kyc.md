# 가입 방법 및 KYC 본인인증

{% hint style="info" %}
📌 KYC 인증은 입출금 서비스 이용을 위해 필수입니다.
{% endhint %}

---

## Step 1 — 공식 홈페이지 접속
[xlence.com/ko](https://www.xlence.com/ko) 접속 후 **'계좌 개설'** 클릭

## Step 2 — 회원 정보 입력
- 이메일 주소 입력
- 비밀번호 설정
- 국가: **South Korea** 선택
- 이용약관 동의 후 **'계속'** 클릭

## Step 3 — 이메일 인증
가입한 이메일로 발송된 **인증 링크** 클릭

## Step 4 — KYC 본인인증
1. 신분증 종류 선택 (여권 / 주민등록증 / 운전면허증)
2. 신분증 앞면 사진 업로드
3. 셀피(selfie) 촬영 및 업로드
4. 제출 후 승인 대기 (보통 수 분~수 시간)

{% hint style="warning" %}
✅ KYC 승인 완료 후 입출금 및 거래가 가능합니다.
{% endhint %}
