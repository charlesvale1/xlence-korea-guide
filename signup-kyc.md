# 가입 방법 및 KYC 본인인증 방법

{% hint style="info" %}
📌 아래 절차를 순서대로 진행해주세요. KYC 인증 완료 후 입출금이 가능합니다.
{% endhint %}

---

## Step 1 — 회원가입
![](https://raw.githubusercontent.com/charlesvale1/xlence-korea-guide/main/image/xlence_kyc_1.png)
[xlence.com/ko](https://www.xlence.com/ko) 접속 후 **'계좌 개설'** 클릭

가입 양식을 **영문**으로 작성 후 **'계좌 개설'** 버튼 클릭

| 항목 | 입력 내용 |
|------|---------|
| 이름 / 성 | 영문 이름 입력 |
| 휴대폰 번호 | +82 선택 후 번호 입력 |
| 메일 주소 | 이메일 주소 입력 |
| 비밀번호 | 소문자·대문자·숫자·기호 포함 6~15자 |
| 거주 국가 | 대한민국 선택 |

✅ 개인정보 동의 체크박스 클릭 후 **'계좌 개설'** 버튼 클릭

---

## Step 2 — 로그인
![](https://raw.githubusercontent.com/charlesvale1/xlence-korea-guide/main/image/xlence_kyc_2.png)
가입 완료 후 로그인 페이지에서:

- **ID**: 가입 시 입력한 이메일 주소
- **암호**: 가입 시 설정한 비밀번호

입력 후 **'로그인'** 클릭

---

## Step 3 — 이메일 및 모바일 인증
![](https://raw.githubusercontent.com/charlesvale1/xlence-korea-guide/main/image/xlence_kyc_44.png)
![](https://raw.githubusercontent.com/charlesvale1/xlence-korea-guide/main/image/xlence_kyc_6.png)
로그인 후 우측 상단 **≡ 메뉴** 클릭 → **Profile → Account** 클릭

| 항목 | 방법 |
|------|------|
| Email | ✏️ 아이콘 클릭 후 이메일 인증 진행 |
| Mobile Phone | ✏️ 아이콘 클릭 후 모바일 인증 진행 |

인증 완료 시 **Verified** 표기 확인

---

## Step 4 — 신원 정보 입력
![](https://raw.githubusercontent.com/charlesvale1/xlence-korea-guide/main/image/xlence_kyc_8.png)
**≡ 메뉴** → **Profile → Profile Information** 클릭

**Personal details** 우측 ✏️ 클릭 후 정보 입력:

| 항목 | 입력 내용 |
|------|---------|
| Full Name | 영문 이름 |
| Full Name (Local) | 한글 이름 |
| Gender | 성별 |
| Date of Birth | 생년월일 |

{% hint style="warning" %}
⚠️ **주소 입력 시 반드시 신분증(주민등록증 또는 운전면허증)에 표기된 주소로 입력해야 합니다.**
{% endhint %}

---

## Step 5 — 주소 입력
![](https://raw.githubusercontent.com/charlesvale1/xlence-korea-guide/main/image/xlence_kyc_9.png)
**Address details** 우측 ✏️ 클릭 후 영문 주소 입력:

| 항목 | 예시 |
|------|------|
| Address line 1 | 407, Wangsan-ro |
| Address line 2 | 동·호수 |
| Postal Code | 우편번호 5자리 |
| State / Region | OO-gu |
| City | Seoul |
| Country | Republic of Korea |

{% hint style="info" %}
💡 영문 주소는 **구글 또는 네이버 영문 주소 입력기**를 활용하면 쉽게 기입 가능합니다.
{% endhint %}

---

## Step 6 — Financial Profile 입력
![](https://raw.githubusercontent.com/charlesvale1/xlence-korea-guide/main/image/xlence_kyc_10.png)
**Financial Profile** 우측 ✏️ 클릭 후 아래 예시 참고하여 입력:

**Tax Residency (세금 거주지)**

| 항목 | 예시 입력값 |
|------|-----------|
| Country of Tax Residence | Republic of Korea |
| TIN | 주민번호 전체 입력 |

**Employment Details (직업 정보)**

| 항목 | 예시 입력값 |
|------|-----------|
| Employment Status | Self Employed |
| Source of Funds | Saving/Investments |
| Nature of Business | FinancialServicesOther |
| Expected deposit (12개월) | $50,000 - $100,000 |

**Financial Details**

| 항목 | 예시 입력값 |
|------|-----------|
| Estimated Annual Income | $50,000 - $99,999 |

---

## Step 7 — Trading Experience 입력
![](https://raw.githubusercontent.com/charlesvale1/xlence-korea-guide/main/image/xlence_kyc_11.png)
**Trading Experience** 섹션에서 거래 경험 입력 (예시 참고):

| 항목 | 예시 입력값 |
|------|-----------|
| Forex and CFD's — Frequency | Occasionally |
| Forex and CFD's — Amount | $50,000 - $100,000 |
| Securities — Frequency | Rarely |
| Securities — Amount | $10,000 - $20,000 |
| Other Derivatives — Frequency | Rarely |
| Other Derivatives — Amount | $10,000 - $20,000 |

{% hint style="info" %}
📋 위 항목은 예시입니다. 본인 상황에 맞게 입력하세요.
{% endhint %}

---

## Step 8 — 신분증 업로드 (KYC)
![](https://raw.githubusercontent.com/charlesvale1/xlence-korea-guide/main/image/xlence_kyc_12.png)
**Profile Documents** 섹션 → **Identity Document** 우측 👁️ 아이콘 클릭

- 신분증 종류 선택: **여권 / 주민등록증 / 운전면허증** 중 선택
- 사진 업로드 진행

{% hint style="warning" %}
⚠️ **신분증 또는 운전면허증 업로드 시, 주소도 반드시 해당 신분증에 기재된 주소와 동일하게 입력해야 합니다.**

추후 주소지 인증도 동일한 신분증 사진으로 진행하면 됩니다.
{% endhint %}

업로드 완료 후 심사 통과 시 **Approved** 표기 확인 ✅
